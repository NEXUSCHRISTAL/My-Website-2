My First Project: Weather Outfit App

October 1, 2025

I just launched my first project, the Weather Outfit App! This app helps you choose outfits based on the weather in your city. It uses real-time data from the OpenWeather API to give you outfit suggestions that match current conditions.

I'm excited about this project because it combines my love for coding with practical everyday use. I plan to add more features soon, like auto-detection of your location and more outfit options.

Digital Clock Project

October 9, 2025

I'm thrilled to share my latest coding project, a Digital Clock that not only tells time but also changes themes based on the time of day. This project has been a fun challenge, pushing me to learn more about JavaScript and CSS animations.

Currently, the clock features themes that shift from day to night. My next goals include adding user-selectable themes like Halloween, fantasy, and cyber themes, as well as an analogue clock display.

The Self Between: Wake Me Gently

December 15, 2025

I wrote a book titled "Wake Me Gently", which is under an ungoing book collection titled "The Self Between". I painted a cover for it as well. But i dont think i like how it turned out.

I tried to see how it would look with a different color tone. But unfortunately, i still dont like the outcome of the piece. The focus was ment to be on the eyes which had a lot of regret and sadness in them. But i feel like the colors took away from that focus instead of adding to it.

The red toned piece was closer to my original vision for the cover art but it is still lacking.

For now, i am just going to postpone the release of the book until i can make a cover that i am satisfied with.

If you are interested in reading the book once its released, please contact me through the contact page. Or you can stay tuned here for updates. I will post updates about the book and its release date.
