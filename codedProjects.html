Weather Outfit App

This app recommends outfits based on the weather. When a user types in their city, it fetches real-time weather data from the OpenWeather API and generates an outfit recommendation suitable for those conditions. I am currently working on adding auto-detection for the user's location.

Technologies used: HTML, CSS, JavaScript, OpenWeather API
